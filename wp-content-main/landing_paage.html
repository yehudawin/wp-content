Required CSV Format

Your CSV file must include these columns for the system to work properly

CSV File Example

Title Domain User Pass CategoryID
Welcome Post https://example.com admin 1234abcd 5
About Us https://mysite.net editor qwerty12 3
Services Page https://blog.org author pass123! 8

Column Descriptions

Title

The title of your blog post. The AI will generate content based on this title.

Domain

The full URL of your WordPress website (including https://).

User

WordPress username with publishing privileges.

Pass

WordPress password for the specified user.

CategoryID

The numeric ID of the WordPress category where the post should be published. You can find this in your WordPress dashboard under Categories.
